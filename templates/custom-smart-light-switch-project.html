custom smart light switch

Description

Created smart light switch using an Arduino UNO with an IR sensor and
servo and a Raspberry Pi with an IR transmitter to "manually" turn the
light switch on and off. The internet connected Raspberry Pi allowed for
remote control over the internet. The light switch was also able to be
controlled via mobile device through SSH commands. This system was later upgraded to an off the shelf Wi-Fi light switch flashed with custom
firmware to integrate with Home Assistant.
